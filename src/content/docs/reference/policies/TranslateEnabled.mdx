---
title: "TranslateEnabled"
description: "Enable or disable webpage translation."
category: "Content settings"
---

Enable or disable webpage translation.

> [!NOTE] Web page translation is performed entirely client-side, so there is no data or privacy risk transmitting content to a translation service over the network.

If you only want to disable the popup, you can set the `browser.translations.automaticallyPopup` preference to `false` using the [Preferences](/reference/policies/preferences) policy.

**Compatibility:** Firefox 126\
**CCK2 Equivalent:** N/A\
**Preferences Affected:** `browser.translations.enable`

## Windows (GPO)

```
Software\Policies\Mozilla\Firefox\TranslateEnabled = 0x1 | 0x0
```

## Windows (Intune)

OMA-URI:

```url
./Device/Vendor/MSFT/Policy/Config/Firefox~Policy~firefox/TranslateEnabled
```

Value (string):

```xml
<enabled/> or <disabled/>
```

## macOS

```xml
<dict>
  <key>TranslateEnabled</key>
  <true/> | <false/>
</dict>
```

## policies.json

```json
{
  "policies": {
    "TranslateEnabled": true | false
  }
}
```
