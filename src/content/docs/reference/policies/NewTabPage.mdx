---
title: "NewTabPage"
description: "Enable or disable the New Tab page."
---

Enable or disable the New Tab page.

**Compatibility:** Firefox 68, Firefox ESR 68\
**CCK2 Equivalent:** N/A\
**Preferences Affected:** `browser.newtabpage.enabled`

## Windows (GPO)

```
Software\Policies\Mozilla\Firefox\NewTabPage = 0x1 | 0x0
```

## Windows (Intune)

OMA-URI:

```url
./Device/Vendor/MSFT/Policy/Config/Firefox~Policy~firefox/NewTabPage
```

Value (string):

```xml
<enabled/> or <disabled/>
```

## macOS

```xml
<dict>
  <key>NewTabPage</key>
  <true/> | <false/>
</dict>
```

## policies.json

```json
{
  "policies": {
    "NewTabPage": true | false
}
```
